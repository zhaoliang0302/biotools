# siRNA 转染体系计算器

这是一个面向实验室日常使用的轻量网页工具，用来快速计算 Lipofectamine 3000 的 siRNA 转染配制体积，并生成中文实验记录文本。

## 核心用途
- 按孔数与分组快速汇总 A 管（Lipo 稀释液）与 B 管（siRNA 稀释液）配制体积。
- 支持“分组富余 + 总管富余”的双重损耗补偿，减少移液不足的风险。
- 一键生成中文操作记录，直接复制到电子实验记录本。

## 使用方式
1. 打开 `index.html`。
2. 选择实验规格或手动调整单孔用量。
3. 设置分组与孔数，必要时调整富余量。
4. 点击“生成方案”，查看配制明细与记录文本。

## 重要说明
- 工具默认只做体积汇总，不自动推导 siRNA 浓度或终浓度。
- 单孔用量应以你的实验条件为准（例如细胞密度、培养基体积、试剂批次等）。
- 若需要“siRNA 原液浓度 + 目标终浓度”的自动计算，可在此基础上扩展。

## 目录结构
- `index.html`：页面结构
- `style.css`：样式
- `script.js`：计算逻辑与实验记录输出
